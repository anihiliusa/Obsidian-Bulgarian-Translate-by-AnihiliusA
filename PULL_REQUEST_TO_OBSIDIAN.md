# Submit to Obsidian

Official repository: `obsidianmd/obsidian-translations`

Target file: `translations/bg.txt`

Workflow:

1. Create your own copy of the official translation repository.
2. Generate `translations/bg.txt` with this toolkit.
3. Validate the generated file.
4. Commit the Bulgarian translation update on a new branch.
5. Open a pull request to the official repository.

Suggested PR title:

```text
Update Bulgarian translation
```

Suggested PR body:

```text
Bulgarian endonym: български език

This PR updates the Bulgarian UI translation for Obsidian.
Translation prepared by AnihiliusA.
```
